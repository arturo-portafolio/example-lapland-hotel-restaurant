# Lapland Hotel Restaurant

A modern hotel & restaurant landing page demo built with React and a clean UI component system.

## Live Demo
https://example-lapland-hotel-restaurant.vercel.app/

## Tech Stack
- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Run Locally
Requirements: Node.js + npm (or bun)

```sh
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev


## Deployment
Deploy with Vercel: import this repository, then Vercel will build and publish automatically on every push to the main branch.

## Custom Domain
In Vercel: Project → Settings → Domains → add your domain and follow the DNS instructions.
